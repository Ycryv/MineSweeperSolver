# 扫雷概率计算器 
[立即使用](https://ycryv.github.io/MineSweeperSolver/)
通过枚举所有未知格子计算精确概率。

### 注意事项
- 棋盘最大支持20x30，非前沿格过多可能会导致卡死
- 填入的**数字0**并不表示周围雷数为0，而是标记为**已知非雷格**
### 🎯 精确概率计算
- 枚举所有可能的雷分布
- 计算每个格子的精确概率
- 支持**已知剩余雷数**和**未知剩余雷数**两种模式

### 概率可视化
- 格子右上角显示概率百分比
- **颜色标识**不同概率区间
  - 0%：亮绿色背景
  - 100%：亮红色背景
  - 其他：暖色渐变
- 已翻开数字用标准扫雷颜色显示

### 交互设计
- **双模式输入**：
  - 先选格子再点数字
  - 先选数字再连续填数
- **键盘导航**：WASD/方向键移动选中
- **实时反馈**：选中状态高亮显示

## 🔍 算法原理

1. **约束提取**：从数字格提取雷数约束
2. **回溯搜索**：深度优先搜索所有可能的雷分布
3. **可行性剪枝**：提前判断部分分配是否满足约束
4. **概率统计**：汇总所有有效布局，计算每个格子的雷概率
